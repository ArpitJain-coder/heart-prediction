# ❤️ Heart Disease Prediction App

This project is a machine learning-powered web application built with Streamlit that predicts the likelihood of heart disease based on user-provided medical parameters.

## 🚀 Features
- Predicts heart disease risk in real-time  
- Interactive and user-friendly interface  
- Input patient health metrics (age, cholesterol, blood pressure, etc.)  
- Displays prediction results instantly  
- Lightweight and easy to deploy  

## 🧠 Machine Learning
The model is trained on a heart disease dataset using classification algorithms such as:
- Logistic Regression  
- Decision Tree / Random Forest  
- Support Vector Machine (optional)

## 📊 Input Parameters
- Age  
- Sex  
- Chest Pain Type  
- Resting Blood Pressure  
- Cholesterol Level  
- Fasting Blood Sugar  
- Resting ECG Results  
- Maximum Heart Rate Achieved  
- Exercise-Induced Angina  
- ST Depression (Oldpeak)  

## 🛠️ Tech Stack
- Python  
- Streamlit  
- Scikit-learn  
- Pandas  
- NumPy
